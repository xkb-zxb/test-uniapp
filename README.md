## 项目基于HBuilderX创建的uni-app项目（内置uni-ui的项目模板）
自带uni架构基础组件和uni-ui扩展组件，其中还有部分可用icon，
然后ColorUI中也有部分icon,
小程序的话，还会本地化引入vant的方式扩展UI，这里也会带有icon
若都不能满足，则再从iconfont上下载png图标
